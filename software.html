Geometry + Simulation Modules (Gismo)

G+Smo (pronounced gismo or gizmo) is a C++ library for isogeometric analysis (IGA). Geometry plus simulation modules aims at the seamless integration of Computer-aided Design (CAD) and Finite Element Analysis (FEA).

Implementations:

  • Structural analysis toolbox with (non-)linear static, dynamic, modal and (post-)buckling analysis modules (gismo/gsStructuralAnalysis @ GitHub)
  • Kirchhoff-Love shell module with nonlinear hyperelastic materials (gismo/gsKLShell @ GitHub)
  • Several improvements in assemblers, quadrature rules and mapped splines (gismo/gismo @ GitHub)
