Geometry + Simulation Modules (Gismo)

G+Smo (pronounced gismo or gizmo) is a C++ library for isogeometric analysis (IGA). Geometry plus simulation modules aims at the seamless integration of Computer-aided Design (CAD) and Finite Element Analysis (FEA).

Implementations:

  • Structural analysis toolbox with (non-)linear static, dynamic, modal and buckling analysis modules
  • Post-buckling analysis modules
  • Kirchhoff-Love shell module
  • Nonlinear material module for thin shells
