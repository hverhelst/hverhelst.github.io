MSc. Projects

Maritime Technology, Mechanical Engineering

1. Experimental investigations of wrinkling formation in membranes with stiffened regions

  • Design and production of testing aparatus,
  • Application of Digital Image Correlation (DIC) as measurement technique,
  • Speciment design and testing.

2. Shape Optimization of Membranes with Integrated Solar Panels

  • Find a suitable objective function for wrinkling reduction (e.g. principal stress-based),
  • Assessment if simplified methods for wrinkling modelling (e.g. with Tension Field Theory),
  • Case-study design and application to a real-life example,
  • Verification with ANSYS or ABAQUS,
  • Use of Isogeometric Analysis in G+Smo (C++); mainly as a blackbox

3. Smooth Multi-Patch Analysis for Conventional Maritime Structures

  • Assessment of various patch-coupling methods for smooth shells w.r.t. computational efficiency, hot-spot localization,
  • Seamless integration of the method with CAD,
  • Verification ANSYS/ABAQUS,
  • Using existing library G+Smo (C++).

Applied Mathematics

4. Physics-Informed Neural Networks (PINNs) for Isogeometric Shells

  • Training of a PINN for isogeometric shells with various boundary conditions, load cases,
  • Integration of the method in Computer-Aided Design (CAD) for interactive evaluation of solutions,
  • Optional: Coupling with deterministic Isogeometric Kirchhoff-Love Shell element,
  • Using existing library G+Smo (C++).

5. Beam modelling using Isogeometric Analysis

  • Implementation of the Euler-Bernoulli beam model in Isogeometric Analysis,
  • Assessment of domain coupling techniques for beams,
  • Modeling of large structures,
  • Using existing library G+Smo (C++).
